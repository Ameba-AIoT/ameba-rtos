The script file is used to send image to device based on UART by OTA function.

Command : 
	python uart_user_ota.py

(1) Please ensure that the version of Python is at least v3.0. The test python version is v3.8.0.
(2) Copy the ota_all.bin image into the directory where the script is.
(3) Modify the port as your actual serial port.
