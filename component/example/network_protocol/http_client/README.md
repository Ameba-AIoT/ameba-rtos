# Example Description

In this example, http client send request to get server response.

# HW Configuration

None

# SW configuration

1. Change `HOST_NAME` and `HOST_PORT` to user define.

2. How to use:
   - Run `./build.py -a http_client` under project path, e.g. `amebasmart_gcc_project/`, to generate images.
   - `Download` images to board by Ameba Image Tool.

# Expect result

This example will call `http_client()` to send http request.
